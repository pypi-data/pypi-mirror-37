# Python Registry Manager
======================= 

---- 

## Installation 
Use pip: 
```
pip install pyrm 
```
if you want to install from source code , you can download from pypi or simple use: 
```
git clone https://github.com/lfzark/pyrm 
```
then run: 
```
python setup.py install 
```
