# Python Registry Manager
======================= 

---- 

## Installation 
Use pip: 
```
pip install faster 
```

if you want to install from source code , you can download from pypi or simple use: 
```
git clone https://github.com/lfzark/faster
```
then run: 
```
python setup.py install 
```

## Example 

```
faster use douban

```
